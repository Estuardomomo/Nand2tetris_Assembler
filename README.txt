Directions for the execution of the program:
1. Press the enabled button to load a source file. This is characterized by the folder icon.
2. A second button with a green icon will be enabled. This button must be pressed in order to 
generate a .hack file which will be located in the same folder as the source file. This new 
file is written in machine language.
